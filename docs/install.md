# Installation Guide

## English

### Download

Download the full `.run` installer from:

- [GitHub Releases](https://github.com/LanceLeeA/router-plugin-hub/releases)

### Recommended Package Type

This project publishes a full upstream-style `.run` installer package.

### Install On Soft Routers

For compatible soft-router environments, this `.run` package is intended to support direct upload and installation through iStore manual installation workflows.

### Notes

- Source code stays in git
- Large installer binaries should be distributed through Releases
- Use the latest release package whenever possible

## 中文

### 下载位置

完整 `.run` 安装包请从下面地址下载：

- [GitHub Releases](https://github.com/LanceLeeA/router-plugin-hub/releases)

### 安装包类型

本项目发布的是完整上游风格的 `.run` 安装包。

### 软路由安装方式

对于兼容的软路由环境，这个 `.run` 安装包可用于在 `iStore` 中直接上传并安装。

### 说明

- Git 仓库保存的是源码、打包脚本和发布说明
- 大体积 `.run` 安装包建议放在 Releases 分发
- 安装时优先使用最新发布版本
