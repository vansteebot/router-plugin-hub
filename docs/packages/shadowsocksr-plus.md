# ShadowsocksR Plus

This page is the package landing note for the `ShadowsocksR Plus` branch of this repository.

- `packages/ssrplus-enhanced`

## What It Contains

- LuCI interface enhancements
- async apply and rebuild behavior
- safer restart and hard cleanup controls
- TXT-based `ss://` import helpers
- auto-switch tuning and monitoring
- IPv6 mode control with safe defaults
- full `.run` packaging scripts

Use this package branch when publishing, maintaining, or extending the enhanced `ShadowsocksR Plus` router package.
